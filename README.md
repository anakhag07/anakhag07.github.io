# Anakha Ganesh - Personal Website

Single-page personal website designed for GitHub Pages.

## Structure

- `index.html`
- `styles.css`
- `script.js`
- `assets/` (headshot and resume PDF)

## Local preview

```bash
python3 -m http.server
```

Then open `http://localhost:8000`.

## Deploy on GitHub Pages

1. Push this repo to GitHub.
2. In GitHub, go to Settings > Pages.
3. Select "Deploy from a branch" and choose `main` / root.
4. Visit the URL GitHub provides (user site: `https://<username>.github.io/`).
